Projeto Titanic¶

Este projeto faz parte do curso de Fundamentos de Data Science I, ministrado pela Udacity, e  pr-requisito para a obteno do diploma no mesmo. 
De acordo com a Wikipdia: "O RMS Titanic foi um navio de passageiros britnico operado pela White Star Line e construdo pelos estaleiros da Harland and Wolff em Belfast. Sua construo comeou em maro de 1909 e ele foi lanado ao mar em maio de 1911. O Titanic foi pensado para ser o navio mais luxuoso e mais seguro de sua poca, gerando lendas que era supostamente inafundvel.
Ele colidiu com um iceberg s 23h40min do dia 14 de abril e afundou na madrugada do dia seguinte com mais de 1 500 pessoas a bordo, sendo um dos maiores desastres martimos em tempos de paz de toda a histria."    
No projeto utilizaremos um dataset, conjunto de dados, que contm informaes de 891 dos 2.224 passageiros e tripulantes a bordo do Titanic. Dados estes que mostram se o tripulante sobreviveu ou no ao desastre, qual era a classe em que viajava, idade, sexo entre outros.
Para realizar a anlise dos dados em questo utilizaremos o notebook Jupyter com Python 3 e suas bibliotecas Pandas e NumPy.

Hipóteses¶

1 - A classe em que o indivduo viajava poderia ser fator determinate para sua sobrevivncia?
2 - O vnculo familiar seria capaz de influenciar, de alguma maneira, pela busca de sobrevivncia?
3 - A idade do indivduo teria algum impacto em suas chances de sobrevivncia?

Estas sero as hipteses investigadas nesta anlise. E para o primerio momento, faremos um levantamento dos dados para verificar a necessidade de limpeza dos mesmos.

#Funcao utilizada para retornar os valores agrupados pela coluna informada.
def agrupaPorColuna(coluna):
    return ds_titanic.groupby([coluna]).count()['PassengerId']

#Funcao utilizada para retornar os valores de sobreviventes pela coluna informada.
def totalSobreviventesAgrupados(coluna):
    return ds_titanic.query("Survived == 1").groupby([coluna]).count()['PassengerId']

#Funcao utilizada para realizar o calculo de percentual
def buscaPercentual(valor1, valor2):
    return (valor1 / valor2) * 100

#Funcao utilizada para plotar os graficos de pizza com os valores repassados
def plotaGraficoPizza(plt, sizes, explode, labels, titulo):
    fig1, ax1 = plt.subplots()
    ax1.pie(sizes, explode=explode, labels=labels, autopct='%1.1f%%', shadow=True, startangle=90)
    ax1.axis('equal') 
    
    plt.title(titulo)
    plt.show()
    
#script para grafico de barras encontrado em https://www.vooo.pro/insights/guia-basico-para-plotar-graficos-usando-o-matplotlib-do-python-parte-2/
